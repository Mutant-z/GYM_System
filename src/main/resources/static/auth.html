这个页面直接对接当前 Spring Boot 认证接口，用来联调会员登录、管理员登录、获取当前登录用户和退出登录。

会员登录 调用 POST /api/auth/member/login，返回 token 与当前用户信息。
管理员登录 调用 POST /api/auth/admin/login，验证角色型登录链路。
当前用户 自动带上 Bearer Token 调用 GET /api/auth/me。
退出登录 调用 POST /api/auth/logout，验证 Redis 中 token 删除。
